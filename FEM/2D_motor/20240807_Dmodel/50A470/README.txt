アキマ補間データの詳細		作成者：岩田和久	作成日：2022/12/20

[データ説明]
・nyu_B^2.csv
1列目：B^2
2列目：ν

・akima_coef.csv
1列目：p0
2列目：p1
3列目：p2
4列目：p3

[データ使用方法]
・処理
if Bi <= B < B
	del = B-Bi
	ν = p0 + p1*del + p2*del^2 + p3*del^3
	dν/dB^2 = p1 + 2*p2*del + 3*p3*del^2

注意：
磁束密度の二乗値がデータ外の場合は以下の二通りの方法で補間する.

1. 真空の透磁率で補間
2. 最終点の勾配を用いて一次関数で算出

2の場合は上の処理を以下の通りに行う.（アキマ補間の定義ではp1は点Biの一次導関数と同値である.）

del = B-Bi
ν = p0 + p1*del
dν/dB^2 = p1


以上.
